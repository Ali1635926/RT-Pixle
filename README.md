## 📖 Overview

**RT Pixle** is a professional pixel art editor built with Python and PyQt5. It combines professional-grade design tools with cutting-edge AI capabilities, including smart palette generation, photo to pixel art conversion, auto-outlining, auto-shading, and intelligent symmetry.

---

## ✨ Features

### 🎨 Design Tools
- Pencil, Eraser, Bucket, Line, Rectangle, Ellipse, Eyedropper, Text
- Adjustable brush size (1-16) and opacity
- Shape fill mode for rectangles and ellipses
- Horizontal/Vertical mirroring

### 🧩 Layer System
- Add, delete, duplicate, and reorder layers
- Layer opacity control
- Toggle layer visibility
- Professional compositing

### 🔄 Undo/Redo
- 40-step history
- Full layer state preservation

### 🤖 AI-Powered Tools
- **Smart Palette Generator** - 6 harmony schemes (Complementary, Analogous, Triadic, Tetradic, Pastel, Monochrome)
- **Photo to Pixel Art** - Convert any image with color quantization
- **Palette Extractor** - Extract dominant colors from images
- **Auto Outline** - Generate perfect silhouettes
- **Auto Shading** - Directional lighting with 8 light directions
- **Smart Symmetry** - Horizontal, Vertical, Both, Diagonal, 180° rotation

### 💻 Advanced Features
- AI Chatbot with natural language commands
- Terminal for power users
- Dark/Light themes
- Full keyboard shortcuts

### 📤 Export Options
- **PNG** with adjustable scale
- **JPG** with quality control
- **HTML5** interactive gallery
- **Project (.rtp)** with full layer preservation
